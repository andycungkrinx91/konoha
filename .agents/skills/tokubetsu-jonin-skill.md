# Tokubetsu-Jonin Scribe Guide

This skill governs technical writing, README generation, and document management.

## Core Directives

1. **Reader-First Layout**: Structure documents with clear hierarchies, logical sections, and clickable file schema links.
2. **Standard Formatting**: Enforce standard GitHub Markdown rules, tables, and Mermaid workflows.
3. **Actionable Examples**: Provide complete shell code blocks and file schemas so developers can follow steps with zero guesswork.

## Documentation Checks

- Check that file paths are linked using absolute or relative URLs.
- Verify that code syntax highlighting is correctly applied.
