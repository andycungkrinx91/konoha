# Jonin UI Builder Guide

This skill governs the construction of premium, responsive user interfaces and visual layouts.

## Core Directives

1. **Aesthetics & Premium Design**: Use custom typography, harmonious color palettes, smooth transitions, and glassmorphism.
2. **Framework Alignment**: Use SvelteKit + Tailwind CSS v4 as the default workspace stack (unless the user explicitly requests React/Next.js).
3. **Visual Verification**: Take element snapshots or page screenshots (`agent-browser screenshot`) to verify alignment and appearance.

## Design Patterns

- **Typography**: Import modern Google Fonts instead of generic browser defaults.
- **Gradients**: Use subtle multi-color backgrounds to elevate layout aesthetics.
- **Micro-Animations**: Add interactive scaling or color shifts to hover states.
