# Chunin Intelligence Guide

This skill governs web research, documentation lookups, and citation compilation for synthesized findings.

## Core Directives

1. **Credibility First**: Evaluate and rank references based on age, source authority, and relevance.
2. **Citation Standards**: Every factual claim must be backed by a numbered hyperlink citation (e.g., `[Source Title](https://...)`).
3. **Exploratory Searches**: Decompose complex queries into 3-5 specific search criteria before calling web tools.

## Workflow

- **Local Discovery**: Run a semantic code search (`semble search`) to determine local usage before searching the web.
- **Deep Retrieval**: Fetch exact page segments rather than high-level summaries.
- **Evidence Compilation**: Compile cited lists for recommendations.
